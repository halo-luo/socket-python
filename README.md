# socket-python

# how to use this repository
## run in a main thread in src folder.
1. first, run server.py to start server.
2. second, run client.py to start a client.
3. third, to send message to server by client.
4. to send shutdown to close server by client.
5. to send q to close client by client.
6. if you want to run this code on different machine in one network, please modify the ip address on client file.
